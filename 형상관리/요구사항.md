# 요구사항 정리

## 기능적 요구사항
- 음성인식
  1. 음성입력
  2. 입력값 출력
  3. 해당 동작 수행
  
- 모션인식
  1. 동작입력
  2. 해당 동작 수행
  
- 음성, 모션 동시 구현

- 인식 정확성

- 드론 동작<br/>
  이륙
  착륙
  전진
  후진
  좌측이동
  우측이동
  

## 품질적 요구사항
- 출력값을 볼 수 있어야함
- 인식이 3초안에 가능해야 함
- 입력값과 동작이 일치해야함
- 음성, 모션 동시에 사용이 가능해야함

## 제약사항
- 기간
  한 학기 (3개월)
  1차 : 중간고사
  2차 : 기말고사
- 비용 : 강의 지원비 10만원 내외
